📱 2. Screen-by-Screen Component Specifications 🎬 Module 0. Splash & Onboarding (오프닝 및 가입) Screen 0.1: Splash (첫 인상) Visual Structure: * #0A0D1A 퓨어 다크 배경 중심에 미니멀한 배치.

Component & Animation:

DNA 이퀄라이저: 이중 나선형 궤적을 그리며 촘촘히 서 있는 세로 막대그래프(Histogram) 픽셀들이 음악 이퀄라이저처럼 위아래로 리드미컬하게 바운스(Bounce)됨.

로고 결합: 막대 조각들이 회전하며 영롱한 3D 입체 투명도를 가진 DNA 나선 로고로 부드럽게 병합(Merge)됨. 로고 표면에는 브랜드 코어 그라데이션 광택 레이어가 흐름.

Typography: 로고 하단에 서서히 페이드인(Fade-in)되는 감성 텍스트.

"내가 없는 순간에도, 너를 가장 나답게 사랑할 또 하나의 나." Twin.me

Screen 0.2: Couple Matching (초대 및 연결) Visual Structure: 토스(Toss) 스타일의 미니멀 인터페이스 레이아웃. 극도로 정돈된 인풋 필드와 라운드 버튼 배치.

Component & UI Interaction:

초대코드 입력 폼: 인풋 포커싱 시 테두리에 미세한 네온 바이올렛 #7C3AED 그로우(Glow) 효과 발생.

DNA 자석 결합 이펙트: 상대방 코드 매칭이 성공하는 순간, 화면 좌우에 끊어져 있던 두 개의 나선 실루엣 일러스트가 화면 중앙으로 스냅(Snap)되며 자석처럼 탁 붙는 그래픽 연출. 동시에 스마트폰 전체에 묵직한 햅틱 커플 결합 진동 피드백 제공.

Screen 0.3: Data Ingestion (말투 학습 가이드) Visual Structure: 상단에 인스타그램 릴스 스타일의 숏폼 가이드 영상 레이어가 재생되며, 하단에는 전처리 진행 상태를 나타내는 프로그레스 컴포넌트 배치.

Component & UI Interaction:

안심 가이드 배너: 개인정보 보호 안내 문구는 자칫 딱딱해 보이지 않도록 불투명도 70%의 블러(Frosted Glass) 카드로 감싸 하단에 고정.

시그니처 드립 카드: 파싱 및 학습이 완료되는 순간, 화면 위로 미끄러지듯(Slide-up) 올라오는 3개의 카드 팩 UI.

카드의 시각 디자인: #1E293B 슬레이트 배경 위에 유저의 최고 빈도수 드립 단어가 그라데이션 볼드 텍스트로 강조 표출됨. (ex. "ㄹㅇㅋㅋ" 카드)

Component 1.2: 실시간 상태 바 (X Trend UI) Visual & Typography: * 엑스(X)의 실시간 트렌드 화면을 오마주한 좌우 슬라이드형 텍스트 칩 레이아웃.

Component 1.3: 추억 링 (Instagram Story Highlight UI) Visual & Typography: * 화면 하단에 가로 방향 스크롤(Horizontal Scroll)이 가능한 원형 컴포넌트 배치.

인스타그램 스토리 하이라이트처럼 원형 테두리에 트윈 그라데이션 링이 감싸져 있어 터치하고 싶게 유도. 터치 시 화면 중앙에 오버레이 팝업창이 뜨며 당시 데이트 사진과 OOTD 메타데이터가 카드 형태로 노출됨.

Micro-Interaction (말풍선 타이핑 호흡):

AI방 내부에서 말풍선 수신 시, 서버 통신이 끝나도 즉시 출력하지 않고 글자 수에 비례한 시간차 가변 딜레이(0.5초~1.5초)를 적용함.

그동안 말풍선 위치에 세 개의 점이 위아래로 부드럽게 움직이는 '서영 AI가 말하는 중...' 로딩 애니메이션을 구현하여 인간다운 타이핑 호흡 연출.

UI Control (연애 초기 모드 토글): * 채팅방 우측 상단 내비게이션 바 영역에 미니멀한 슬라이더 토글(Toggle Switch) 배치. 활성화 시 테두리가 그라데이션 컬러로 채워지며 내숭 모드 발동 상태임을 인지시킴.

Component 2.2: 말투 교정 Bottom Sheet Visual Structure: * AI방에서 특정 말풍선을 롱 프레스(Long Press)할 때, 배경화면이 어두워지며(Dim) 화면 최하단에서 매끄럽게 슬라이드 업(Slide-up)되는 바텀 시트 구조.

Component & Hierarchy: * 최상단에 시트를 아래로 쓸어내릴 수 있는 미니멀한 핸들 바 가이드라인 배치.

내부에는 [너무 다정함], [너무 딱딱함], [유머 코드 안 맞음] 등 직관적인 텍스트 라디오 버튼 배치. 선택 완료 버튼 클릭 시 시트가 아래로 완전히 닫히고, 해당 말풍선 텍스트 내용이 부드러운 플립(Flip) 애니메이션과 함께 수정된 답변으로 실시간 교체됨.

Component 2.3: 위기 감지 전면 팝업 (Crisis Mode) Visual Structure: * 앱 사용 중 혹은 푸시 클릭 시 화면 전체를 강렬하게 장악하는 풀스크린 오버레이 뷰.

Design & Effect: * 배경 전체에 사이렌 레드(#EF4444) 컬러가 은은하게 점멸(Pulse)하는 비주얼 효과 레이어 배치.

중앙에는 국가 재난문자 스타일의 타이포그래피 경고창 배치: 🚨 [Twin.me 연애 전선 위기 경보].

출력과 동시에 재난문자 특유의 '두우- 두우-' 진동 주기를 마이크로 세컨드 단위로 쪼개어 약하게 재현한 햅틱 패턴 구동. Center 하단에는 두 에이전트가 도출한 해결책 화면으로 진입하는 대형 그라데이션 액션 버튼 배치.

View 1: 뷰 파인더 모드 (지도형 UI)

네이버/카카오 지도 API 프레임 기반. 커플이 등록한 장소 핀(Pin) 위에 해당 날짜의 서영/세준 사진이 미니 원형 크롭 형태로 얹어짐. 핀 터치 시 하단에 영업시간과 메뉴 가격이 포함된 글래스모피즘 텍스트 카드가 스르륵 오픈됨.

View 2: DNA 나선 모드 (3D 스크롤 타워)

비주얼: 중앙 수직 기둥을 나선형으로 휘감아 올라가는 입체적 램프(Helix Ramp) 그래픽 인터페이스.

레이아웃 규칙: 타워의 최하단(뿌리)은 '처음 사귄 날', 위로 스크롤하여 탑으로 올라갈수록 최신 축의 데이트가 배치되는 역발상 구조. 스크롤 휠 이동 시 카메라 시점이 나선을 타고 3D 회전하며 입체적으로 상승/하강하는 시각 제어 적용.

감성 필름 이펙트: 각 데이트 사진 카드 옆에 투명한 아세테이트 필름 재질의 서브 레이어(The Sentiment Ribbon)가 레이어드됨. 카톡 동기화 시 추출된 그날의 베스트 다정 대사 텍스트가 만년필 폰트로 각인되어 있으며, 스크롤 시 빛을 받아 반짝이는 셰이더 효과 적용.

Component 3.2: 데이트 코스 셔틀 팝업 (무물 스티커 인터페이스) Visual Structure: 인스타그램 스토리의 인터랙티브 스티커(무물 창, 설문조사 스티커) 디자인 완벽 오마주.

Component & Animation: * 파스텔 및 그라데이션이 적용된 둥글둥글한 그래픽 아이콘 버블들이 화면에 통통 튀며 등장.

음식 종류, 오늘 데이트 무드 탭 칩을 선택하는 인터랙션 레이아웃. 특히 '오늘 서로의 OOTD' 컴포넌트는 캐주얼/정장/시밀러룩 아이콘 리스트로 구성하여 터치 피드로 입력 장벽 제거. 결과 도출 시 마법 빗방울이 내리는 듯한 그래픽 이펙트와 함께 최적 동선 카드가 큐레이션됨.

⚙️ Module 4. Settings Tab (설정 및 스토어) Component 4.1: 마이 트윈 데이터 센터 & 지우개 UI Visual Structure: 메타(Meta)의 계정 센터와 유사한 독립된 화이트/다크 슬레이트 카드 그룹 레이아웃. 카드 내부 디자인은 선 요소를 최소화하고 여백을 활용해 가독성 확보.

Component & Control:

프라이버시 슬라이더: 1단계(찐사랑), 2단계(밀당), 3단계(비밀) 영역이 명확히 분할된 수평 트랙 슬라이더 UI. 노브(Knob)를 드래그할 때 각 단계의 텍스트 타이틀이 볼드 처리되며 배경의 그라데이션 채도가 달라지는 시각 피드백 연출.

기억 삭제 지우개: 학습된 데이터 리스트 옆에 배치된 미니 지우개 아이콘. 유저가 아이콘 터치 시, 해당 대화 요약 열 행(Row) 전체가 포토샵에서 지우개로 문질러 지워지듯 투명하게 가루가 되어 흩날리며 사라지는(Dissolve) 애니메이션 이펙트 적용.

Component 4.2: 비즈니스 스토어 배너 및 상품 카드 Visual & UI Control: * 설정 탭 중단에 큼직하게 배치된 프리미엄 스토어 진입 구역. 배너 백그라운드는 네온 바이올렛(#7C3AED)에서 피치 핑크(#FF6B8B)로 흐르는 그라데이션 레이어가 아주 느린 속도(ex. 4초 주기)로 은은하게 그라데이션 위치를 바꾸며 숨쉬는(Breathing) 듯한 고급 셰이더 렌더링 적용.

구독 플랜 카드 (Coffee Break / Deep Talk Night): 하단 상품 리스트 카드는 기본적으로 투명도 60%의 프로스เต드 글래스(Frosted Glass) 효과로 블러 처리되어 매혹적인 호기심 유도. 결제 완료 시 블러 셔터가 위로 스르륵 열리며 내부의 컬러풀한 에이전트 애니메이션 카드 활성화.
